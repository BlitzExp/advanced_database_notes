An **INNER JOIN** returns only the rows that have matching values in both tables. A **LEFT JOIN** returns all rows from the left table and the matching rows from the right table (with `NULL` if there is no match). A **RIGHT JOIN** does the opposite, returning all rows from the right table and matching rows from the left. A **FULL JOIN** (or FULL OUTER JOIN) returns all rows from both tables, matching where possible and filling with `NULL` where there is no match. The term **OUTER JOIN** is a general category that includes LEFT, RIGHT, and FULL joins, since they all return unmatched rows in addition to matched ones.
